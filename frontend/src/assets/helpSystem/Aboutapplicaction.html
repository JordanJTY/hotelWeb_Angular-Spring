About application


       Our Ako hotel application is a website where you can order reservations, in case you are a user, and manage the hotel, in case you are an administrator. To do this, once you start the application you will find yourself in the main menu of the application where you can log in or view the information provided. Once logged in, you will be able to book an apartment, review your reservations and even edit them, being able to change the date in case the starting day of the reservation is not the current one at that moment. An option has been included that allows users to delete their account, use it with caution, because nothing will be preserved once the action is taken. 


                       



               




       For the administrator section, you will find the charts of the different apartment types along with a button at the top of the page that will allow you to add new apartment types. In addition, the administrator will have access to the booking record of all users with relevant booking information. The administrator will be able to delete them in case he/she considers that the reservation is not correct or has encountered a problem with it.
